The size of dots indicators can be small, default, or large depending on how subtle or prominent the indicator should be. Spacing of the dots can be either comfortable, cozy, or compact depending on the availability of space on the page.

If interaction is enabled, the user is able to navigate to the selected step by selecting the corresponding dot indicator. Once the dots are in focus, navigation can be accomplished using the keyboard arrow keys.
