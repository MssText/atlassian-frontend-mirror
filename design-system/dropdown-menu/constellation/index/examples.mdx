## Appearance

### Default

Use `default` for the default dropdown menu appearance. The default menu will scroll after its height exceeds the pre-defined amount.

<Example
  Component={DropdownMenuDefault}
  packageName="@atlaskit/dropdown-menu"
/>

### Tall

Use `tall` to control the height of the menu. The tall menu will not scroll until the height exceeds the height of the viewport.

<Example Component={DropdownMenuTall} packageName="@atlaskit/dropdown-menu" />

## Custom triggers

There are two recommended ways to customize a trigger.

### Using trigger

Set `trigger` to a custom react component which accepts the provided props.

<Example
  Component={DropdownMenuCustomTriggerButton}
  packageName="@atlaskit/dropdown-menu"
/>

Use the `trigger` prop and pass in an html element.

<Example
  Component={DropdownMenuCustomTrigger}
  packageName="@atlaskit/dropdown-menu"
/>

Use the `trigger` prop and pass in a string.

<Example
  Component={DropdownMenuDefault}
  packageName="@atlaskit/dropdown-menu"
/>

## Open by default

Use `defaultOpen` to make the menu open by default.

<Example
  Component={DropdownMenuDefaultOpen}
  packageName="@atlaskit/dropdown-menu"
/>

## States

### Loading

If `isLoading` is true, a spinner is rendered instead of the dropdown items.

<Example
  Component={DropdownMenuLoading}
  packageName="@atlaskit/dropdown-menu"
/>

### Open

Use `isOpen` to control the open state of the dropdown menu.

<Example Component={DropdownMenuOpen} packageName="@atlaskit/dropdown-menu" />

## Positioning

### Default positioning

By default the dropdown menu will be positioned next to your trigger.

### Position

Use `position` to set the menu position, for example bottom right.
