## Tree items 

### Terminal

Set `hasTerminal` to display a terminal on the drop indicator. Use a terminal to help disambiguate between hierarchical changes in trees.

```
<DropIndicator hasTerminal edge="bottom">
  {({className}) => (
    <TreeItem className={className} />
  )}
</DropIndicator
```

### Inset

Use `inset` to align the drop indicator with an items nesting level.

```
<DropIndicator inset={32} hasTerminal edge="bottom">
  {({className}) => (
    <TreeItem className={className} />
  )}
</DropIndicator
```
